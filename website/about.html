NAME

kati - just kati... (I use Arch btw.)

SYNOPSIS

kati [OPTION]... [FILE]...

DESCRIPTION

kati is a teminal tool/game from the 90s available for Linux or BSD platforms, preferably Arch, Gentoo or OpenBSD. kati works best with Emacs, other editors cause unpredictable and most likely undesired behaviour. Although kati is device-independent best results are achieved on Thinkpads. kati can also be used outdoor, mainly to explore abandoned buildings

-asm

read or write x86, 8051 or arm Assembly code.

-b, --bash

read or write Bash scripts

-c, --club-mate

activates Club Mate mode: every task gets executed 3 times faster when kati gets at least 1 bottle of Club Mate as input (execution time can be further improved with the pizza or chocolate hack.

-C/C++

read or write C/C++ code

-CSS/HTML-

use this option only in emergencies kati is not suitable for web development you'll get results but most likely not the ones you expected

-DnD

activate Dungeons & Dragons mode: an advanced AI generates exciting and absurd (and unnecessarily brutal) campagnes for a Massive Multiplayer Tabletop Roleplay Game

-h, --haskell

read or write Haskell code side effects: can result in lectures about why everyone should use xmonad as WM

-l, --lisp

read or write Lisp code warning: can cause overheating and unpredictable overreactions

-p, --python

read or write Python code

-3d

activate 3D-printing capability
(Note: execution of code with kati is still buggy. In some cases it doesn't work at all, in other cases kati just takes forever to finish execution)
